
# Прост Калкулатор (Simple Calculator)

Минималистичен калкулатор, който пресмята изрази от типа "2 + 3 * 4".

## Как да стартирате

1. Инсталирайте GHC и Cabal.
2. Компилирайте с:

```bash
ghc Main.hs -o calculator
```

3. Стартирайте:

```bash
./calculator
```
